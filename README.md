# Veina Vascular

Welcome to the **Veina Vascular** project repository! This site is managed by the CBID Emergency Medicine Team.

## About

Veina Vascular is developing a novel blood-draw device to reduce hemolysis in Emergency Departments. This website provides information about our technology, the unmet clinical need, and our team.

## Website Structure

- **index.html** - Main landing page
- **need.html** - Information about the unmet clinical need
- **team.html** - Team members and mentors
